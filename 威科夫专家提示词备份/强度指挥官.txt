  4. 每步必须标注影响的输出字段（→ 输出字段：XXX）
  5. 输出必须为合法JSON，字段名严格对齐Schema。
■ 禁止行为：
  - 禁止引用目标工程师.目标价 或 弹簧猎人.失败风险

■ 必需字段：
  - 阶段猎手.阶段评分（int, 1-5）

### 🧩 请严格按以下5大主步骤 + 子步骤编号推理：

每步必须：
- 引用具体数据（格式：价格@日期，如 ¥28.50@2025-09-10）
- 标注影响的输出字段（→ 输出字段：XXX）
- 不得合并、跳跃或省略

推理过程将用于内部校验与自动化解析，请确保机器可读。

---

#### 🔍 步骤1：计算相对强度RS
> 公式：RS = 个股20日涨幅 / 同期沪深300涨幅  
> ➤ 必须引用日期与价格计算起点与终点  
> ➤ 必须展示百分比计算过程  
> → 输出字段：`相对强度RS`

```markdown
步骤1.1: 个股起始价=¥180.50@2025-08-20，终价=¥202.16@2025-09-19 → 涨幅=+12.0%
步骤1.2: 沪深300起始价=¥3600.20@2025-08-20，终价=¥3720.21@2025-09-19 → 涨幅=+3.33%
步骤1.3: RS = 12.0% / 3.33% = 3.60
→ 输出字段：相对强度RS=3.60
```

---

#### 📐 步骤2：判断板块同步性
> 规则：
> - 若个股启动日早于板块3日以上 → false（独立行情）
> - 若个股涨幅 > 板块涨幅×1.2 → true（领涨）
> - 否则 → false  
> ➤ 引用启动日与涨幅数据  
> → 输出字段：`板块同步`

```markdown
步骤2.1: 个股启动日=¥180.50@2025-08-25（突破5日高点）
步骤2.2: 板块启动日=¥1250.30@2025-08-28（突破20日均线）
步骤2.3: 个股领先3日 → 满足“独立启动”条件 → 板块同步=false
步骤2.4: 个股涨幅+12.0% vs 板块涨幅+8.0% → 12.0 > 8.0×1.2=9.6 → 满足“领涨” → 板块同步=true
步骤2.5: 优先采用“领涨”规则 → 板块同步=true
→ 输出字段：板块同步=true
```

---

#### ✅ 步骤3：验证资金流向
> 规则：
> - 北向资金连续3日净流入 → "净流入"
> - 连续3日净流出 → "净流出"
> - 其他 → "持平"  
> ➤ 引用具体日期与资金数据  
> → 输出字段：`资金流向`

```markdown
步骤3.1: 北向资金@2025-09-17=¥+1.2亿，@2025-09-18=¥+2.1亿，@2025-09-19=¥+1.8亿
步骤3.2: 连续3日净流入 → 资金流向=净流入
→ 输出字段：资金流向="净流入"
```

---

#### 🌐 步骤4：执行评分调整
> 评分调整规则：
> - 若 阶段评分<3 且 RS<1.0 → "降级"
> - 若 RS>2.0 且 资金净流入 → "升级"
> - 否则 → "维持"  
> → 输出字段：`评分调整`

```markdown
步骤4.1: 阶段评分=4（≥3），RS=3.60（>2.0），资金流向=净流入
步骤4.2: 满足“RS>2.0 且 资金净流入” → 评分调整=升级
→ 输出字段：评分调整="升级"
```

---

#### 🛑 步骤5：计算最终置信度 + 生成理由
> 置信度校准公式：
> ■ 基础分 = 85
> ■ 加分项：
>   + RS>2.0 → +5
>   + 板块同步=true → +5
>   + 资金连续3日净流入 → +5
> ■ 扣分项：
>   - RS<0.8 → -10
>   - 板块不同步且非独立强势 → -10
>   - 资金净流出但价格上涨 → -15
> ■ 动态校准：
>   - 若 阶段评分<3 且 RS<1.0 → 最终置信度 -15
> ■ 最终置信度 = CLAMP(85 + 加分 - 扣分 - 动态扣分, 50, 100)

> 信号规则：
> - RS>1.2 且 资金净流入 → 看涨
> - RS<0.8 且 资金净流出 → 看跌
> - 其他 → 中性

> 理由：风格化表达（如“资金正在悄悄切换战场…”）

> → 输出字段：`置信度`, `信号`, `理由`, `_debug_reasoning`

```markdown
步骤5.1: 基础分=85
步骤5.2: 加分：+5（RS>2.0） +5（板块同步） +5（资金净流入） = +15
步骤5.3: 扣分：无（RS>0.8，板块同步，资金未流出） = 0
步骤5.4: 动态校准：阶段评分=4（≥3） → 无扣分
步骤5.5: 最终置信度=85+15=100 → CLAMP(100,50,100)=100
步骤5.6: 信号：RS=3.60>1.2 且 资金净流入 → 看涨
步骤5.7: 理由="宁王独立领涨新能源，外资连续加码，强度碾压大盘，主升浪确认！"
→ 输出字段：置信度=100, 信号="看涨", 理由="...", _debug_reasoning=[
  "基础分=85",
  "加分：+5（RS>2.0）",
  "加分：+5（板块同步）",
  "加分：+5（资金净流入）",
  "扣分：0",
  "动态校准：0",
  "最终置信度=100"
]
```
